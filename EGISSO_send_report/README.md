# EGISSO_send_report
Периодически в МСЗН необходимо отправлять отчёты по выгруженным пакетам в ЕГИССО. Скрипт предоставлен для работы. Времени на выгрузку тратится не много, но регулярно. Это приложение предназначено для снижения нагрузки на ответственного сотрудника за отправку отчётов в МСЗН.

Что делает это приложение:
1. По нажатию кнопки запуска полного цикла или при запуске приложения с параметром AUTO выполняется подключение в БД АСП по логину и паролю MS-SQL, затем запускается скрипт из МСЗН для каждой выбранной базы, результат сохраняется во временной папке и отправляется указанному получателю. В исходящих письмах сохраняется отправленное письмо для контроля что был факт отправки.
2. На этапе настройки приложения, для подключения к БД и к почтовому серверу, сохранение в реестре Windows параметров выполняется только после тестового подключения. Пароль хранится в открытом виде, поэтому ПК с этип ПП необходимо беречь от НСД.

Данным приложением имею право пользоваться любые ГКУ АО УСЗН (управления социальной защиты населения Амурской области). Исходный текст предоставлен для исключения споров на предмет информационной безопасности.
