# check_people_at_base
Проверка людей из списка (по СНИЛС) с базами АСП. Добавляет коментарии для тех строк, которые найдены в базе АСП.

Создано для снижения нагрузки у специалистов при проверке реестров от СФР и закрытия заявок в АСП гражданам, которыеполучают МСП через СФР.

Создано в "Embarcadero RAD Studio XE7\Delphi XE7"
